# 武堂

## 基本信息
- 全称：青阳郡武堂
- 直属上级：青阳郡府（郡县独立武备机构，非修仙宗门附属）
- 驻地：青阳郡城东
- 职能：追剿尸僵、辖区内治安、承接郡县委托任务
- 人员规模：约30人（队长1名·副队长2名·资深侍卫8-10名·普通侍卫约20名）
- 成员境界：以炼体期为主，队长田猛为练气初期

## 组织架构
- **队长**：田猛（练气初期）——统领全局，分配任务，对接郡府
- **副队长**：待定 —— 带领外出任务，对接郡府联络
- **资深侍卫**：石万鸿、白常等人 —— 执行任务骨干，可带新人
- **药师**：赵太和 —— 掌管药堂，负责救治和日常炼血丹炼制
- **普通侍卫**：日常巡逻、追剿尸僵、协助资深侍卫

## 炼血丹经济体系

### 来源

炼血丹由两条渠道供给：

| 渠道 | 来源 | 品级 | 用途 |
|------|------|------|------|
| 药堂自炼 | 赵太和以药堂丹炉炼制 | 普通炼血丹 | 月俸配发、日常淬体 |
| 郡县炼药署 | 郡县官办炼药机构，以委托报酬形式拨付 | 普通炼血丹（质量更稳定） | 补充药堂储备、任务奖励 |

- 郡县炼药署隶属于青阳郡府，由郡府财政供养，设专职炼药师数名，为郡县下属各武备机构提供丹药
- 药堂自炼的丹药每月约产出30-40枚，郡县拨付不定量（视委托任务量而定，约20-30枚/月）
- 两者均为普通炼血丹，品质相当。进入练气期后需改用灵石及更高品丹药，炼血丹退场

### 郡县委托制度

武堂与青阳郡为委托关系，非上下级。郡县向武堂下发任务委托，武堂完成后郡县以炼血丹或贡献物资作为报酬。

| 任务类型 | 报酬 |
|---------|------|
| 清除普通尸僵 | 2-3贡献点/具 |
| 清除异化尸僵 | 10-15贡献点/具 |
| 辖区内治安巡逻 | 1贡献点/次 |
| 协助郡县押运/护送 | 2-5贡献点/次 |

### 贡献点体系

贡献点是武堂内部核算单位，不消耗，仅用于兑换。与任务积分不同——积分决定等级晋升，贡献点用于兑换实物。

**贡献点兑换表**：

| 兑换项目 | 所需贡献点 |
|---------|----------|
| 1枚炼血丹 | 5点 |
| 功法抄本（基础） | 10点 |
| 功法抄本（进阶） | 30点 |
| 兵器维护/修复 | 2-5点 |
| 药膏、绷带等耗材 | 1点 |
| 赵太和特制疗伤药 | 8点 |

**贡献点来源**：仅通过完成委托任务获取。月俸中的贡献点是武堂根据侍卫等级每月固定划拨的基础值，无需任务即可获得。

### 月俸表

| 级别 | 实物配额 | 贡献点/月 | 折合总丹数 | 备注 |
|------|--------|---------|--------|------|
| 见习弟子 | 1枚 | 0点 | 1枚 | 见习期三个月，期满转正 |
| 普通侍卫 | 2枚 | 6点 | ~3枚 | 见习期满后 |
| 资深侍卫 | 3枚 | 10点 | ~5枚 | 需出师考核 |
| 副队长 | 3枚 | 15点 | ~6枚 | — |
| 队长（练气） | 4枚 | 30点 | ~10枚 | — |

- 兑换价：1枚炼血丹 = 5贡献点
- 见习弟子无贡献点——未出任务，无法获得任务报酬。满三个月且至少参与一次任务后方可转正
- 队长田猛的贡献点显著高于他人，因其承担对接郡府和郡县的额外职责

### 炼血丹用途

| 用途 | 说明 |
|------|------|
| 淬体修炼 | 服用后配合站桩/运功引导药力，淬炼筋骨皮膜。是炼体期最核心的修炼方式 |
| 疗伤吊命 | 重伤时可用于打通经脉、护住心脉。轻伤涂抹丹粉即可 |
| 解瘴气毒 | 仅限凡人中毒——武者对瘴气毒抗性较高，炼血丹可辅助化解。修士无效 |

### 民间流通

- 郡县药铺有售，但价格昂贵——约合普通百姓数月收入。仅富户或武备人员家属可负担
- 黑市偶有流通，武堂禁止侍卫私下转卖。违者扣减配额
- 普通百姓基本接触不到——炼血丹在凡人眼中是"武者专用的金贵东西"

## 炼体境界

武堂侍卫所修为通用炼体体系，共九层，分前中后三期，每三层为一期。第十层为破限层，非天赋异禀或奇遇加身不可触及。

| 阶段 | 层级 | 淬炼重点 | 战力参照 |
|------|------|---------|---------|
| **前期** | 第1层·炼皮 | 皮肤韧化，外力难伤 | 普通侍卫入门 |
| | 第2层·锻肉 | 肌肉纤维增粗，力量倍增 | 普通侍卫标准 |
| | 第3层·强筋 | 筋腱强化，爆发力质变 | 普通侍卫进阶 |
| **中期** | 第4层·壮骨 | 骨骼致密如铁 | 资深侍卫入门 |
| | 第5层·活髓 | 骨髓造血加速，恢复力跃升 | 资深侍卫标准 |
| | 第6层·淬脏 | 五脏强化，耐力惊人 | 资深侍卫进阶 |
| **后期** | 第7层·换血 | 血液渐有灵性，感知力初现 | 副队长级 |
| | 第8层·通脉 | 经脉贯通，可短暂外放气血 | 副队长顶尖 |
| | 第9层·凝窍 | 周身三十六窍穴初凝，距引气仅一步 | 准练气 |
| **破限** | 第10层·破限 | 超越常规极限，肉身通灵 | 筑基之下无敌 |

**尸僵对战参照**：
- 普通尸僵 ≈ 炼体第3-4层肉身强度，但无痛觉、无章法——需2-3名普通侍卫合力击杀，资深侍卫单杀
- 异化尸僵 ≈ 炼体第7-8层，且带有阴寒残留气息——仅田猛（练气期）能以境界压制击杀

## 关键人物行为记录

### 田猛
- 关键行为：批准从个人配额支取炼血丹，与石万鸿共同救治大眼；仅他一人可独立斩杀异化尸僵
- 人物特质：果断但不鲁莽，对下属严苛但护短，对青牛村事件有隐约怀疑
- 结局：第一卷中期发现泣血门献祭真相后被练气期弟子灭口

### 石万鸿（资深侍卫）
- 身份：资深侍卫，田猛手下六年，沉默寡言、行事稳重
- 关键行为：在青牛村废墟下用个人炼血丹救下大眼，后续丹药全部从自己配额补足
- 人物背景：三年前带队追剿尸僵时，因月初耗尽丹药无法救治队友洛平。此后三年每月只服两枚丹、留一枚。
- 人物特质：寡言、沉稳，对下属严厉但对弱小者有天然的守护欲
- 后续定位：大眼的引路人之一

### 白常
- 关键行为：初期对耗费丹药救外人持保留态度，后被田猛和石万鸿说服。田猛安排他教导大眼
- 人物特质：务实、有些小算计但不坏，嘴上抱怨但做事认真
- 后续定位：大眼的武道启蒙师傅

### 赵太和
- 身份：药堂药师，掌管炼丹与救治
- 职能：日常炼血丹炼制（每月产出30-40枚）、伤员救治、药堂管理
- 人物特质：沉默寡言，四十年老药师，手上茧子比脸上的皱纹还厚
- 备注：武堂药堂的丹炉为小型丹炉，仅能炼制普通炼血丹。郡县炼药署配有大型丹炉和专业炼药师团队

## 世界观备注
- 炼血丹是炼体期最基础的修行丹药，也是武堂经济运转的核心
- 进入练气期后炼血丹退场——改用灵石辅以高品淬体丹药
- 陈凡/大眼因修习《泣血炼体大法》，突破路径与通用炼体九层有本质区别，但仍需消耗大量炼血丹辅助淬体
