# 武学手段大全

> 本文档汇总三修体系下的全部武学手段——剑诀、刀法、雷法、精神攻击、毒、防御。每个手段标注所属体系、品阶和对应境界。

---

## 一、剑诀类

### 泣血·伤情剑法

| 属性 | 说明 |
|------|------|
| **所属体系** | 泣血炼体大法（身） |
| **品阶** | 成长型剑法——随境界提升而解锁新式 |
| **设定** | 泣血门的入门剑法，几乎人人都会。但大部分人只练到前两式便放弃了——因为从第三式开始，剑招的威力与自身的痛苦程度成正比。痛得不够深的人，永远练不出剑意。陈凡以其特殊的炼体根基，悟出了这套剑法的真谛：这不是剑法，是把痛化为剑的艺术。 |

#### 五式详解

**第一式·断肠**
- 境界：炼体前期即可
- 招式：直刺，快而狠，剑出无悔
- 特点：入门式——剑速快过同阶剑法三成，但无附加属性
- 疼痛代价：轻微刺痛（可忽略）

**第二式·碎心**
- 境界：炼体中期（泣血皮膜后）
- 招式：斜斩，以自身心口剧痛为引，增幅剑气
- 特点：斩出的剑气会在空中留下短暂的"血痕"——这是泣血之力外放的雏形
- 疼痛代价：中度——每次出剑心口如被刺穿

**第三式·离魂**
- 境界：炼体后期（熔炉玉脏后）
- 招式：回旋斩，剑势飘忽，斩人神魂
- 特点：剑招不再是纯物理攻击——中剑者会短暂眩晕，仿佛魂被斩离了身体
- 疼痛代价：重度——出剑时全身经脉如被撕裂

**第四式·绝命**
- 境界：炼体圆满（泣血圣体后）/ 练气期
- 招式：舍身一击，燃烧精血，同归于尽的剑招
- 特点：战力瞬间提升至当前境界的极限——代价是一剑之后经脉尽断
- 疼痛代价：极限——出剑时的痛苦超越身体承受上限，靠"魔心道骨"硬抗

**第五式·无伤**
- 境界：筑基期后（理论要求·陈凡在特殊状态下提前悟出）
- 招式：大圆满后的反璞归真——剑出无痛，因为痛已内化为力量本身
- 特点：不再需要"感受痛"来驱动剑招——痛觉已经成为他身体的一部分，像呼吸一样自然
- 代价：无需主动激发痛苦——但需要前四式的万次以上挥剑才能达到此境

### 紫霄·天罚九剑

| 属性 | 说明 |
|------|------|
| **所属体系** | 紫霄神雷经（神） |
| **品阶** | 天阶剑法（残篇——缺少最后三式，需第二卷中自行参悟或寻找完整传承） |
| **设定** | 将雷霆之力凝于剑上，每一剑都如天劫降临。这是上古雷修的最高剑道传承——以剑引雷，以雷代天。 |

#### 六式详解

**第一剑·雷动**
- 对应雷法阶段：引雷（练气期）
- 招式：快剑，如雷霆一闪，电光火石间取人性命
- 特点：出剑速度超越同阶所有剑法——很多对手还没看清剑身，已经被刺穿了

**第二剑·雷落**
- 对应雷法阶段：引雷（练气期）
- 招式：重剑，引天雷下击，劈在剑尖所指之处
- 特点：远程单体高伤害——需要雷暴天气或自身雷窍中有足够储备

**第三剑·雷网**
- 对应雷法阶段：蕴雷（筑基期）
- 招式：守剑，剑势成网，周身布下雷霆结界
- 特点：防御型剑式——可抵挡同阶的群攻法术，并对近身者施以雷电反击

**第四剑·雷怒**
- 对应雷法阶段：蕴雷（筑基期）
- 招式：暴剑，将体内所有雷霆一次性释放，剑气覆盖百丈
- 特点：AOE大招——范围广但不分敌我，用完后需要时间补充雷霆储备

**第五剑·雷寂**
- 对应雷法阶段：化雷（金丹期）
- 招式：杀剑，中剑者不止肉身被毁，灵魂也被雷霆净化，不入轮回
- 特点：最强单体杀招——对阴邪鬼物有一击必杀的效果，但消耗极大

**第六剑·雷劫**
- 对应雷法阶段：掌雷（元婴期）
- 招式：道剑，以己身代天劫。中剑者将面临自身心魔与雷霆的双重考验
- 特点：这不是物理攻击，是"天道审判"——内心有愧者无法抵御。正道修士中剑后可能反而淬炼道心，邪修中剑则直接魂飞魄散

**第七至九式**：残篇——第二卷主线之一：寻找完整的天罚九剑传承。

#### 蓝星关联
他在雷雨天修炼后回到蓝星，身上还带着残留电荷。赵青瑶碰他时被电了一下，缩手骂他："你是皮卡丘转世吗！"从此这成了两人的专属梗。每次打雷，她都会给他发消息："皮卡丘，你的主场到了。"

## 二、刀法类

### 九幽·斩魄刀

| 属性 | 说明 |
|------|------|
| **所属体系** | 九幽魂典（魂） |
| **品阶** | 禁术级 |
| **设定** | 此刀法不修刀，只修意。以魂力为刀，专斩神魂。刀刃由纯粹的精神力凝成——无实体，却能在灵魂上留下永不愈合的伤口。对战时绕过肉身防御，直接攻击神魂。 |

#### 四式详解

**斩魄·断念**
- 对应魂法阶段：搜魂（练气期）
- 效果：斩断对方当前的念头——可打断正在施法的术式，或让敌人瞬间"忘了自己要做什么"
- 代价：轻微魂力消耗

**斩魄·噬忆**
- 对应魂法阶段：控魂（筑基期）
- 效果：斩中后，可窃取对方某段特定记忆——拷问时最实用的技能
- 代价：中等魂力消耗，且被窃取的记忆会残留在自身神魂中数日

**斩魄·夺志**
- 对应魂法阶段：分魂（金丹期）
- 效果：斩灭对方的战意与求生欲——不是杀死对方，是让对方"不想活了"
- 代价：重度魂力消耗，自身也会短暂陷入抑郁

**斩魄·灭魂**
- 对应魂法阶段：噬魂（元婴期）
- 效果：全力一击，直接抹杀神魂——被斩者连轮回都入不了
- 代价：永久损耗自身部分魂力。非深仇大恨不轻用。

## 三、雷法类

### 掌心雷
- **所属体系**：紫霄神雷经
- **品阶**：基础雷法，但随境界提升威力递增
- **三形态**：
  - 远攻·雷弹：压缩雷霆成球，掷出后爆炸
  - 近战·雷拳：将雷霆缠绕在拳头上，配合泣血魔体近战
  - 控制·雷链：释放连锁闪电，弹射多个目标，麻痹对手

### 雷遁术
- **设定**：身化雷霆，以雷光的速度短距离瞬移
- **进化路线**：
  - 初期（引雷·练气）：瞬移十丈，使用后有短暂僵直
  - 中期（蕴雷·筑基）：瞬移三十丈，轻微僵直
  - 后期（化雷·金丹）：可连续瞬移，雷暴天可借天雷超远距离传送数百里
  - 圆满（掌雷·元婴）：一念即至，短距离无限制

### 雷铠
- **设定**：将雷霆覆盖全身，形成雷电铠甲
- **效果**：近身者被雷电反击，对阴邪属性攻击有极强克制
- **叠加**：与泣血魔体叠加时攻防一体——但消耗巨大

## 四、精神攻击类

### 九幽·搜魂术
- **设定**：读取濒死之人或意志薄弱者的记忆
- **应用**：拷问情报、追查线索、读取敌人战术意图
- **代价**：每次使用后自身神魂轻微受损，被搜者的负面情绪会短暂污染自身
- **剧情位置**：男主用此术追查灭门案线索——每次使用，他都会在敌人瞳孔中看到自己冰冷倒影的瞬间

### 九幽·夺魂眼
- **设定**：将魂力凝聚于双目，对视时可震慑对方神魂
- **效果**：
  - 轻则让低阶修士短暂失神（1-3秒）
  - 重则可直接摧毁同阶以下修士的神智
- **代价**：眼中会累积死气。在蓝星，赵青瑶偶尔会觉得他的眼神突然很陌生

### 泣血·杀意实质化
- **设定**：将泣血魔功积累的杀意与痛苦转化为精神威压
- **效果**：高阶修士可凭杀意直接让低阶修士崩溃
- **特点**：陈凡的杀意中混杂了两界记忆——中招者会看到蓝星与修仙界的碎片画面，心智不坚者会直接疯掉

## 五、毒类

### 泣血·血毒
- **设定**：泣血圣血练成后（炼体第四层·换血重生），自身血液本身就是一种毒
- **机制**：他的血中蕴含泣血魔气和修炼时积累的各种痛苦记忆碎片。敌人若沾染：
  - 伤口无法愈合，持续承受剧痛
  - 被痛苦的记忆碎片反噬——幻象频生
- **应用**：
  - 涂抹武器：让敌人伤口恶化，战力持续下降
  - 血雾：自残喷血形成毒雾困敌——以伤换命的典型打法
  - 血契：让敌人饮下自身血液，种下血毒禁制——违逆则毒发

## 六、防御类

### 魔体·铁壁
- **所属体系**：泣血炼体大法
- **设定**：泣血炼体的被动防御——皮膜如法宝，骨骼如神铁
- **效果**：同阶物理攻击很难破防。是泣血魔功修炼者敢于"以伤换命"的底气
- **进化**：每突破泣血炼体一层，魔体自动强化一次

### 雷铠
- **所属体系**：紫霄神雷经
- **设定**：将雷霆覆盖全身形成雷电铠甲
- **效果**：近身者被雷电反击，对阴邪属性攻击有极强克制
- **组合**：与魔体·铁壁叠加时攻防一体，但体内三系同时运转消耗巨大

### 魂壁
- **所属体系**：九幽魂典
- **设定**：以九幽魂力构筑精神防御壁垒
- **效果**：抵御搜魂、夺舍、幻术等精神攻击
- **特点**：陈凡的魂壁中掺杂了蓝星记忆——入侵者会看到高楼大厦、汽车飞机、手机电脑等完全无法理解的画面，从而产生困惑和破绽。这是他的独家优势
- **雷笼术**（变体）：以法拉第笼原理，以自身雷霆在神魂表面构建一层"电磁屏障"——既保护神魂不被外部入侵，也防止自身雷法灼烧神魂
